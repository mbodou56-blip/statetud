À Propos

🎯 Objectif du projet

StatEtud est une application de collecte et d'analyse descriptive avancée développée en L2 Informatique à l'Université de Yaoundé I.


Secteur ciblé : santé et bien-être étudiant — stress, sommeil, alimentation, activité physique et performance académique.

📐 Statistiques couvertes

  • Moyenne, Médiane, Mode, Asymétrie
  • Écart-type (σ), Variance (σ²), CV%
  • Q1, Q3, IQR, Boîtes à moustaches
  • Corrélation de Pearson (matrice)
  • Régression linéaire (OLS) + R²
  • Test t de Student (une population)
  • Test du Chi² d'indépendance

⚙️ Technologies

HTML5CSS3 JavaScriptPython 3 FlaskSQLite WebSocketsSocket.IO Chart.js 4

🏗️ Architecture Expert

  • Base de données : SQLite (persistant, robuste)
  • Sécurité : Validation serveur + sanitisation XSS
  • Rate limiting : 10 req/min par IP
  • Temps réel : WebSockets (Socket.IO)
  • Export : CSV + Rapport HTML/PDF
  • Stats : Module Python natif (pas de NumPy)
